## 🚀 Learning Highlights

### Mathematical Rigor
Deep dive into the statistical mechanics behind machine learning, focusing on how models estimate parameters through **MLE** and **MAP** to handle bias and variance.

### Industrial Vision & Robotics
As a Robotics student, I focused on applying **CNN** and **Vision Transformer** architectures to bridge the gap between pattern recognition theory and real-world computer vision tasks.

### Modern AI Implementation
Explored the transition from traditional supervised learning to modern **Self-Supervised** and **Generative models**, providing a comprehensive toolkit for building intelligent robotic systems.
